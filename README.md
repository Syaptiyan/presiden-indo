# 🇮🇩 Presiden Indonesia

Profil lengkap 8 Presiden Republik Indonesia dengan timeline interaktif, biografi detail, dan UI/UX premium.

## 🚀 Live Demo

👉 [https://syaptiyan.github.io/presiden-indo](https://syaptiyan.github.io/presiden-indo)

## ✨ Fitur

- **8 Presiden RI** — dari Soekarno hingga Prabowo
- **Biografi Lengkap** — latar belakang, karir, konteks sejarah
- **Detail Pribadi** — lahir, wafat, partai, wakil presiden
- **Masa Jabatan** — durasi lengkap dengan tanggal
- **Pencapaian Utama** — 6+ pencapaian per presiden
- **Timeline Interaktif** — urutan kronologis dengan animasi
- **Badge Kategori** — Proklamator, Reformasi, Demokrasi, dll
- **Dark Theme** — desain modern dan elegan
- **Responsive** — tampil sempurna di mobile dan desktop

## 🛠️ Teknologi

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Glassmorphism)
- JavaScript
- [Alpine.js](https://alpinejs.dev/) — reactive framework
- [Font Awesome](https://fontawesome.com/) — icons
